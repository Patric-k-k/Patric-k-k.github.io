There are 3 arguments that can be passed to the Python-Advanced-Error-Handleing-Package or Error_Handle as we are going to refer to it as, so to import it you would do:

The output of this would be: All arguments are optional, and if you do not pass any arguments it will default to the following: All the arguments are explained below:
Type Of Argument Argument Definition
Code 0-999 Adds 'Error code:' and your error code at the end of your type and Msg
Msg String Just put in a string and it will go after your type!
Type Illegal_Char When you put this in it says: 'Error: Illegal Charector detected.' and it will add on a Msg/Err_Code if you give it one.
Type RGB_Overload When you put this in it says: 'Error: RGB Overload!' and it will add on a Msg/Err_Code if you give it one.
Type RGB_Underload When you put this in it says: 'Error: RGB_Underload!' and it will add on a Msg/Err_Code if you give it one.
Type UNKOWN When you put this in it says: 'An unkown error has ocourred, please restart the program or contact the developer(s) of the application.' and it will add on a Msg/Err_Code if you give it one.
Type Random When you put this in it picks a random one and it will add on a Msg/Err_Code if you give it one.
Type Digit_Overload When you put this in it says: 'Error: Digit_Overload!' and it will add on a Msg/Err_Code if you give it one.
Type No_input When you put this in it says: 'Error: No input detected' and it will add on a Msg/Err_Code if you give it one.
Type Interupted When you put this in it says: 'Error: Operation was Interupted. To try again, please restart the program and wait for it to complete its operation.' and it will add on a Msg/Err_Code if you give it one.
